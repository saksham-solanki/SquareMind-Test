       - "use client" directive
       - Import { useState, useEffect, useCallback } from "react", { motion, AnimatePresence, type PanInfo } from "framer-motion"
       - Props: { testimonials: Array<{ text: string; name: string; role: string }> }
       - State: [activeIndex, setActiveIndex], [isPaused, setIsPaused]
       - Direction tracking for slide animation: track direction (1=forward, -1=backward) to animate slides in from the correct side

       **Auto-advance logic:**
       - useEffect with setInterval(5000) that advances to next testimonial
       - Clear interval when isPaused is true
       - Cleanup on unmount

       **Swipe detection:**
       - Wrap the testimonial content in motion.div with drag="x" and dragConstraints={{ left: 0, right: 0 }}
       - onDragEnd handler: if offset.x < -50 go next, if offset.x > 50 go prev (threshold-based, not velocity -- simpler and more reliable for 3 items)
       - dragElastic={0.2} for slight overshoot feel

       **Pause on hover:**
       - onMouseEnter={() => setIsPaused(true)}, onMouseLeave={() => setIsPaused(false)}
       - Also pause during drag: set isPaused true on onDragStart, false on onDragEnd

       **Animation:**
       - AnimatePresence mode="wait" with custom variants based on direction:
         - enter: { x: direction > 0 ? 80 : -80, opacity: 0 }
         - center: { x: 0, opacity: 1 }
         - exit: { x: direction > 0 ? -80 : 80, opacity: 0 }
       - Transition: { duration: 0.35, ease: "easeInOut" }

       **Layout:**
       - Outer container with overflow-hidden
       - Testimonial card: bg-cream rounded-[20px] p-10 max-lg:p-8 (match existing card style)
       - Stars: text-gold text-[16px] mb-4 tracking-[3px] "★★★★★"
       - Quote: font-serif text-[18px] leading-[1.55] text-ink italic
       - Author: border-t border-cream-dark, name bold, role gray

       - Dot indicators below: flex gap-2 justify-center mt-8, each dot is a button with w-2.5 h-2.5 rounded-full, active: bg-sage, inactive: bg-gray-300, transition-colors, onClick to jump to that index
