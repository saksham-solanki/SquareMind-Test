## What Is Circle Rate?

Circle rate is the minimum property value set by the government for stamp duty calculation. It varies by location, property type, and is updated periodically. If you buy a property below circle rate, stamp duty is still calculated on the circle rate.

## Different Names, Same Concept

| State | Term Used |
| --- | --- |
| Maharashtra | Ready Reckoner Value / ASR |
| Delhi | Circle Rate |
| Karnataka | Guidance Value |
| Telangana | Market Value |
| Gujarat | Jantri |
| Tamil Nadu | Guideline Value |
| UP | Circle Rate |
| Haryana | Collector Rate |

## How Circle Rate Affects You

### Stamp Duty Calculation
Stamp duty = Rate x Max(Transaction Value, Circle Rate)

If you buy at Rs 80L but circle rate is Rs 1Cr, stamp duty is on Rs 1Cr.

### Capital Gains Tax (Seller)
If sale price is below circle rate by more than 10%, the circle rate is deemed as sale consideration for capital gains calculation.

### Income Tax (Buyer)
If purchase price is below circle rate by more than 10%, the difference is taxable as "Income from Other Sources" under Section 56(2)(x).

## Example: Impact on Both Parties

Property with Circle Rate Rs 1Cr, sold at Rs 85L:

**Seller:**
- Sale consideration deemed: Rs 1Cr (circle rate)
- Capital gains calculated on Rs 1Cr, not Rs 85L

**Buyer:**
- Stamp duty on Rs 1Cr
- Difference (Rs 1Cr - Rs 85L = Rs 15L) taxable as income
- Tax on Rs 15L at slab rate

## When Circle Rate Is Lower Than Market

In fast-growing areas (Noida Extension, Dwarka Expressway), market rates often exceed circle rates by 20-40%. In this case:

- Stamp duty on actual transaction value (higher amount)
- No adverse tax implications
- You are paying the real market price
