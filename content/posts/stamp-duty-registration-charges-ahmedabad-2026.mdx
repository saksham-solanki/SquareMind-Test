## Ahmedabad Stamp Duty Rates (2026)

Gujarat offers some of the lowest stamp duty rates among major Indian states:

| Component | Rate |
| --- | --- |
| Stamp duty | 4.9% |
| Registration charge | 1% |
| **Total** | **5.9%** |

No gender-based concession in Gujarat. Same rates apply to all buyers.

## Jantri Value System

Gujarat uses "Jantri" (government-prescribed property values) as the basis for stamp duty calculation. Updated periodically (last major revision in 2023). Stamp duty calculated on the higher of: actual transaction value or jantri value.

Key Ahmedabad jantri rates:

| Area | Jantri Value (per sq mt) |
| --- | --- |
| SG Highway | Rs 35,000-55,000 |
| Prahlad Nagar | Rs 40,000-60,000 |
| Bodakdev | Rs 38,000-55,000 |
| Bopal | Rs 22,000-35,000 |
| Gandhinagar (GIFT City area) | Rs 20,000-32,000 |

## Online Registration Process

1. Visit garvi.gujarat.gov.in
2. Create account with Aadhaar
3. Enter property details (TP scheme, FP number)
4. System calculates stamp duty based on jantri value
5. Pay stamp duty via e-Stamping
6. Book SRO appointment
7. Complete registration with biometric verification

## Gujarat-Specific Points

1. **TP Scheme:** Town Planning Scheme number is essential for urban Ahmedabad properties
2. **NA permission:** Required for agricultural land conversion
3. **AMC property tax:** Ahmedabad Municipal Corporation tax must be current
4. **GIFT City:** Special regulations may apply — verify with GIFT SEZ authority
