## Why RERA Check Is Your First Step

Since 2017, every residential project with more than 8 units or 500 sqm must be registered under RERA. Buying an unregistered project means zero legal protection under the Act.

## How to Check RERA Registration (Any State)

### Step 1: Identify the State RERA Portal

| State | Portal | URL |
| --- | --- | --- |
| Maharashtra | MahaRERA | maharera.maharashtra.gov.in |
| Karnataka | KRERA | rera.karnataka.gov.in |
| Tamil Nadu | TNRERA | rera.tn.gov.in |
| Telangana | TSRERA | rera.telangana.gov.in |
| Haryana | HRERA | haryanarera.gov.in |
| UP | UP RERA | up-rera.in |
| Gujarat | Gujarat RERA | gujrera.gujarat.gov.in |
| West Bengal | WBHIRA | wbhira.gov.in |
| Delhi | Delhi RERA | rera.delhi.gov.in |
| Goa | Goa RERA | rera.goa.gov.in |

### Step 2: Search the Project

Search by:
- Project name
- Builder/promoter name
- RERA registration number (printed on all marketing material — mandatory)

### Step 3: Verify These Details

| What to Check | Why It Matters |
| --- | --- |
| Registration status | Active, expired, or revoked |
| Registered carpet area | Must match what builder is selling you |
| Registered possession date | The legally binding timeline |
| Promoter details | Builder name and SPV entity |
| Total units and sold units | Sales progress |
| Quarterly progress update | Is construction progressing? |
| Complaints filed | Buyer grievances |
| Bank account details | Escrow account for project funds |

### Step 4: Check Builder's Track Record

Search the builder's name on RERA portal to see ALL their registered projects. Check:
- How many projects registered
- How many delivered on time
- How many have active complaints
- Any revoked registrations

## What RERA Registration Guarantees

| Protected | Not Protected |
| --- | --- |
| Carpet area as registered | Quality of construction |
| Possession date (legally binding) | Price appreciation |
| Builder must use 70% funds for that project | Location-specific risks |
| Right to file complaint with RERA | Builder's financial health |
| Penalty for delays | Market conditions |

## Red Flags to Watch For

1. **No RERA registration** — never buy from an unregistered project
2. **Expired registration** — project may be stalled
3. **Registered carpet area differs from marketed area** — builder inflating numbers
4. **No quarterly updates filed** — possible project stall
5. **Multiple complaints with similar issues** — systemic problems
6. **RERA number not displayed on marketing** — violation of RERA norms
7. **Different SPV name than expected** — verify the exact legal entity

## RERA Complaint Process

If a builder violates RERA terms:
1. File complaint on the state RERA portal
2. Pay nominal fee (₹1,000-5,000)
3. RERA authority investigates and adjudicates
4. Orders are enforceable — can include compensation, interest, and refund

## Frequently Asked Questions

### Is RERA registration enough due diligence?

No. RERA registration is necessary but not sufficient. You still need title verification, encumbrance check, and builder financial health analysis. RERA protects your carpet area and timeline rights but doesn't guarantee title or quality.

### What if a project is not registered under RERA?

Do not buy. If a project requires RERA registration and doesn't have it, the builder is violating the law. You have no RERA protection for complaints, delays, or refunds.
