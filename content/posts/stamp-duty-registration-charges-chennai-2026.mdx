## Chennai Stamp Duty Rates (2026)

Tamil Nadu has one of the highest combined stamp duty and registration rates in India:

| Category | Rate |
| --- | --- |
| Stamp duty (male) | 7% |
| Stamp duty (female) | 6% |
| Registration charge | 4% |
| **Total (male)** | **11%** |
| **Total (female)** | **10%** |

Women buyers save 1% on stamp duty. The 4% registration charge is among the highest in India and significantly impacts total acquisition cost.

## Guideline Value System

Tamil Nadu uses "Guideline Values" set by the Registration Department. Updated every 3-5 years. Published on tnreginet.gov.in. Stamp duty calculated on the higher of: actual sale value or guideline value.

## Online Registration Process

1. Visit tnreginet.gov.in
2. Create account and link Aadhaar
3. Enter property details (survey number, zone)
4. System calculates stamp duty based on guideline value
5. Pay stamp duty online
6. Book SRO appointment
7. Visit for biometric registration with seller

## Key Chennai-Specific Rules

1. **CMDA approved layout:** Only properties in CMDA/DTCP approved layouts get clear title registration
2. **Patta and Chitta:** Must be in seller's name before registration
3. **Encumbrance Certificate:** Available online on tnreginet.gov.in
4. **Property tax:** Chennai Corporation tax must be current before registration

## Cost Reduction Strategies

1. Register in woman's name (saves 1%)
2. Consider emerging areas where guideline values are lower (Perungalathur, Guduvanchery)
3. If guideline value is significantly lower than market rate, you pay on actual value anyway
4. First-time buyer PMAY benefits may partially offset high stamp duty
